# 🎮 AlgoQuest

> A Gamified Algorithm Learning Experience in RPG Style  
> 🏆 Winner – Beginners Track (36hr Hackathon)

---

## 🚀 Overview

**AlgoQuest** is a 2D RPG-style game designed to make learning algorithms fun, interactive, and immersive.

Instead of passively reading theory or solving isolated problems, players:

- 🗡 Battle enemies using algorithms  
- 🧠 Solve logic-based challenges  
- 📈 Level up by mastering Data Structures & Algorithms  
- 🎯 Progress through algorithm-themed quests  

We built AlgoQuest to bridge the gap between **gaming and education**, making algorithm learning engaging rather than intimidating.

---

## 🎯 Problem Statement

Traditional algorithm learning platforms:

- Feel repetitive and non-interactive  
- Lack immersion and storytelling  
- Focus only on problem-solving, not conceptual understanding  

Many beginners struggle because DSA feels abstract and boring.

💡 **AlgoQuest transforms algorithms into combat mechanics and story progression.**

---

## 🕹️ Gameplay Mechanics

- 🧩 Solve algorithm challenges to defeat enemies  
- ⚔️ Different enemies represent different algorithm categories  
- ❤️ Health system based on answer correctness  
- 🎒 Inventory system for power-ups  
- 📊 Progression system with leveling  
- 🗺️ RPG-style exploration  

---

## 🧠 Learning Integration

Each game mechanic is mapped to core CS concepts:

| Game Element | Algorithm Concept |
|--------------|------------------|
| Combat       | Time Complexity |
| Enemy Types  | Sorting, Searching, Graphs |
| Boss Fights  | Advanced Problems |
| XP System    | Concept Mastery |
| Power-Ups    | Optimization Techniques |

---

## 🛠️ Tech Stack

- **Frontend:** JavaScript / HTML / CSS  
- **Rendering Engine:** p5.js  
- **Backend:** Node.js / Express  
- **Database:** MongoDB  
- **Authentication:** JWT / OAuth  
- **Deployment:** Render
